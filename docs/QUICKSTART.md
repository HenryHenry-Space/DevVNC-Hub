# Dev VNC Server

通用开发用远程桌面服务，用于在 SSH 远程连接环境下调试 GUI 应用程序。  
A general-purpose remote desktop service for debugging GUI apps over SSH.

## 功能 / Features

- Xvfb 虚拟显示器 / Virtual display
- x11vnc VNC 服务器 / VNC server
- noVNC Web 访问 / Web access
- 多种窗口管理器支持 / Multiple window managers

## 快速开始 / Quick Start

```bash
# 启动 / Start
dev-vnc start

# 浏览器访问 / Browser access
http://localhost:6080/vnc.html
```
